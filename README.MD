# Jay's blog

技术笔记,啥都写.

## 文章书写
* clone 仓库
```bash
git clone -b write https://github.com/aiyijing/aiyijing.github.io.git
```
* 安装环境
```bash
npm install --force
```
* 新建文章
```bash
hexo new post "xxxx"
```

## 书写要求

* 禁止使用中文标点
* 英文与中文用空格隔开
* 代码必须使用高亮

仓库已经配置 Github Action

* master 禁止 push
* 编写文章从 write 新建分支并编写文章
